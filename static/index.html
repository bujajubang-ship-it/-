▶ 콘텐츠 리서처

촬영 전 시청자 욕구를 먼저 파악하세요

주제 키워드를 입력하면 유튜브 상위 영상 댓글과 네이버 카페를 분석해서
사람들이 뭘 원하는지, 경쟁 영상의 빈틈은 뭔지 알려드립니다.

예시 키워드:

📊 시장 분석 요약
💡 시청자 욕구 분석
🤔 사람들이 궁금한 것
    😤 불만사항 / 페인포인트
      ✨ 원하는 것
        ❓ 시청자가 가장 많이 묻는 질문 TOP 10
          📝 영상에 반드시 넣어야 할 내용
            ⚡ 차별화 포인트 (경쟁 영상의 빈틈)
            🎯 경쟁 영상 분석
            제목 패턴
              썸네일 스타일
                인기 키워드
                📹 상위 경쟁 영상
                🎬 추천 영상 구성

                문제 정의 → 제목 → 썸네일을 확정하세요

                촬영 전에 제목과 썸네일이 먼저 결정되어야 영상 방향이 흔들리지 않습니다.
                키워드와 제품 설명을 입력하면 문제 정의부터 CTR 10%+ 제목·썸네일까지 뽑아드립니다.

                문제 정의 · 제목 · 썸네일 기획안

                🎯 문제 정의
                ✍️ 추천 제목 5개 (CTR 10%+ 목표)
                🖼️ 썸네일 컨셉 3개

                문제제기→공감→손해→이득→사례 도입부를 작성하세요

                5단계 공식으로 30초 안에 시청자를 사로잡는 도입부 대본을 만들어 드립니다.
                기획 탭의 문제 정의 결과를 붙여넣으면 바로 쓸 수 있는 대본이 나옵니다.

                5단계 공식 도입부 대본

                ✍️ 완성 도입부 대본
                📐 단계별 분석
                🎲 대안 훅 (A/B 테스트용)
                  🎥 촬영 팁

                    레퍼런스 대본을 내 영상으로 변형하세요

                    잘 된 영상 대본을 붙여넣으면 구조를 분석하고 내 제품/주제에 맞게 변형합니다.
                    책·전문가·시연 요소 추가 제안과 댓글 유도 마무리까지 포함됩니다.

                    레퍼런스 기반 변형 대본

                    🔍 레퍼런스 구조 분석
                      ✍️ 변형된 전체 대본
                      📐 섹션별 변형 내용
                      📚 추가하면 좋은 요소 (책·전문가·시연 등)
                      💬 댓글 유도 마무리

                      촬영한 영상의 편집 피드백을 받으세요

                      키워드와 영상 대본을 입력하면 시장 데이터와 비교 분석해서
                      어디를 살리고 어디를 삭제할지, 제목과 썸네일은 어떻게 할지 알려드립니다.

                      시장 데이터 기반 편집 피드백

                      📋 전반적 평가
                      ✅ 잘 된 점
                        💚 살려야 할 구간
                        ✂️ 삭제 추천 구간
                        ⚡ 인트로 후킹 피드백
                        🔧 편집 흐름 개선점
                          📌 시청자가 원하는데 빠진 내용
                            ✍️ 추천 제목 (이 영상에 최적화)
                            🖼️ 썸네일 추천
