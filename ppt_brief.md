# PPT 제작 요청 프롬프트

아래 내용을 바탕으로 발표용 PPT를 제작해줘.

---

## 발표 제목
**부자주방 유튜브 콘텐츠 기획 시스템 — 전체 전략 원칙**

## 발표 목적
유튜브 영상을 만들기 전에 내가 고려하는 모든 기획 원칙과 리서치 방법을 정리한 자료.
채널 운영 전략, 제목/썸네일 원칙, 도입부 공식, AI 분석 시스템 전체를 커버.

## 슬라이드 구성 요청
총 15~18장 분량. 각 섹션은 아래 순서대로.

---

## 섹션 1 — 채널 정량 목표 (모든 기획의 기준점)

목표 지표:
- 썸네일 CTR: 10% 이상
- 초반 30초 이탈률: 40% 미만

핵심 메시지: 모든 제목·썸네일·도입부·훅 추천은 이 두 기준을 달성할 수 있도록 역산 설계

---

## 섹션 2 — 채널 & 타겟 정의

- 채널명: 부자주방
- 시청자: 외식업 운영자 (식당·분식집·한식당 사장님), 외식업 창업 준비자
- 제품: 업소용 주방용품 (가스레인지, 업소용 냉장고, 수납용품 등)
- 서비스: 주방 도면설계, 인테리어 시공
- 시청자 특성: 가성비·내구성·사용편의에 민감 / 실용적이고 구체적인 정보 선호 / 40~60대 자영업자

---

## 섹션 3 — 콘텐츠 유형 전략 (풀링 vs 키)

3가지 유형:
1. 풀링 콘텐츠: 외식업·자영업·창업에 관심 있는 넓은 대중을 끌어당기는 정보성 콘텐츠 (조회수 목표)
2. 키 콘텐츠: 시청자의 실제 문제를 제품·서비스로 해결하는 판매 중심 콘텐츠
3. 풀링+키 겸용: 유용한 정보 제공하면서 자연스럽게 제품·서비스 판매까지 (최고 가치)

발행 비율 목표: 풀링 5 : 키 2
풀링+키 겸용 주제를 최우선으로 발굴

예시: "주방동선 설계 꿀팁" = 정보성(풀링) + 도면설계·주방용품 자연 판매(키) → 겸용

---

## 섹션 4 — 제목 작성 원칙 (선택·후회·조건)

클릭률이 높은 제목은 이 3가지 심리 중 하나 이상을 반드시 자극:

- 선택: "이것만 선택하면 됩니다", "두 가지 중 하나만 고르세요"
- 후회: "이걸 모르면 나중에 후회합니다", "안 샀더니 결국 후회했어요"
- 조건: "이 조건이면 무조건 사세요", "이런 분만 보세요"

규칙: 추상적 표현 금지, 시청자 상황을 구체적으로 특정할 것

---

## 섹션 5 — 썸네일 문구 원칙 (결과·감정·공감)

3가지 중심:
- 결과: 구체적 수치·시간·성과 → "매출 2배", "1년 써보니", "이거 하나로 해결됨"
- 감정: 놀람·후회·안도·설렘 → "충격", "반전", "드디어"
- 공감: "맞아, 나도 그래"라고 느끼는 문구 → 흔한 고민·상황을 짧게 압축

전략: 반응률(댓글+좋아요/조회수) 높은 경쟁 영상을 벤치마크 → 부자주방 버전으로 변형 적용

---

## 섹션 6 — 썸네일 이미지 구성 원칙

3가지 원칙:
1. 줌(zoom) 클로즈업: 주제를 화면에 꽉 차게, 배경 보이지 않을 만큼, 시청자 시선이 즉시 핵심으로
2. 시각적 근거 직관적으로: 제품이면 실물, 결과라면 실제 결과물 그대로 (추상적 이미지 금지, 0.5초 안에 무슨 영상인지 알 수 있어야 함)
3. 타겟 맞춤 스타일: 40~60대 식당 사장님 → 깔끔하고 실용적, 실제 주방·제품·사람이 나오는 사진이 효과적

촬영 방법: 앵글·거리·피사체 크기·구도를 구체적으로 명시해서 촬영자가 바로 재현 가능하게

---

## 섹션 7 — 도입부 5대 원칙 (이탈률 40% 미만 달성)

① 썸네일에서 시청자가 기대한 것을 넘어서는 임팩트를 줄 것
② 썸네일에서 언급한 내용을 도입부 30초 안에 반드시 한 번 더 말할 것 → 기대 충족 → 신뢰 형성 → 이탈 방지
③ 시청자가 "맞아, 나도 이 상황이야"라고 강하게 공감하는 장면이나 고민을 먼저 꺼낼 것
④ 클라이맥스 B-roll, 놀라운 결과 장면을 앞으로 당겨서 배치 → 이탈률 감소
⑤ 초반 30초 안에 "이 영상을 끝까지 봐야 하는 이유"를 명확하게 전달

---

## 섹션 8 — 도입부 공식 (5단계 중 상황별 선택 조합)

5단계:
- 문제제기: 시청자가 처한 현실 상황 제시
- 공감: "맞아, 나도 그래" 강한 공감 유발
- 손해: 이걸 모르면/안 하면 생기는 불이익
- 이득: 이 영상을 끝까지 보면 얻는 것
- 사례: 실제 사례·증거·경험

핵심: 전체를 순서대로 쓰는 것이 아니라 영상 유형에 맞게 2~3개를 골라 조합

유형별 추천 조합:
- 정보/꿀팁 → 공감 + 이득
- 신제품 소개 → 문제제기 + 이득 + 사례
- 비교/검증 → 공감 + 손해 + 이득

---

## 섹션 9 — 원고(대본) 욕구 자극 원칙

물건 소개 영상 흐름:
공감 (이 상황 알죠?) → 기존 해결책의 한계 (그래서 다들 이렇게 하는데) → 이 제품의 해결 방식 (그런데 이게 다르게 해결해요) → 구체적 결과 (실제로 ○○ 됩니다) → 상상하게 만들기 ("사장님 주방에 이게 있으면...")

정보 영상 원칙:
시청자가 "이 정보를 나만 알고 싶다", "저장해야겠다"는 느낌이 들도록 구성

모든 원고는 단순 설명이 아닌 감정과 이득을 연결하는 언어로 작성

---

## 섹션 10 — 리서치 데이터 소스 3가지

1. YouTube 데이터 (상위 영상 10개 이상)
   - 조회수·좋아요·댓글수 (반응률 = 댓글+좋아요 ÷ 조회수)
   - 인기댓글 좋아요순 상위 30개 → 시청자의 실제 욕구·불만·궁금증 추출
   - 썸네일 이미지 시각 분석 → 클릭 유도 패턴 추출
   - Most Replayed 히트맵 → 시청자가 집중한 구간 → 핵심 정보 배치 전략 반영

2. 네이버 카페 데이터
   - 식당 사장님 커뮤니티 실제 고민·질문·불만
   - YouTube 댓글로 잡히지 않는 실제 구매 결정 맥락 파악

3. ViewTrap 데이터 (성과율 기반 영상 분석 서비스)
   - 성과 영상: 구독자 대비 조회수 성과율 높은 검증된 영상
   - 핫비디오: 최근 업로드 중 성과율 높은 신규 영상
   - 제목 패턴·썸네일 구성을 부자주방 주제로 변형·적용

---

## 섹션 11 — AI가 분석해서 뽑아내는 결과물

키워드 하나 입력 시 자동 생성되는 기획안:

- 시청자 욕구 분석: 궁금증 / 불만·페인포인트 / 원하는 것 각 5~6개
- 콘텐츠 유형 분류: 풀링+키 / 풀링 / 키 + 자연스러운 판매 연결 각도
- 벤치마크 분석: 반응률 높은 경쟁 영상 선정 → 썸네일 패턴 분석 → 부자주방 버전 변형
- 제목 5개: 선택·후회·조건 중 어떤 심리 전략인지 명시
- 썸네일 3개: 문구 + 줌 피사체 + 촬영 방법 + 색상·분위기 + 클릭 유도 이유
- 문제 정의: 시청자 상황 → 핵심 욕구 → 영상이 해결하는 각도
- 도입부 대본: 선택한 공식 + 썸네일 콜백 + 임팩트 장면 + 완성 대본 (30~50초)
- 전체 대본: 섹션별 4~7개 (타임코드 + 촬영팁 포함)
- 유튜브 설명글 + 인스타그램 캡션: 키워드 8회+ 반복, 자사몰 링크 포함
- CTA 대본: 댓글·구독 유도 구어체

---

## 섹션 12 — 편집 피드백 분석 항목

촬영 후 대본을 AI에 넣으면 자동 분석:

- 시장 적합도 점수 (0~100)
- 살려야 할 구간 vs 잘라야 할 구간
- 도입부 체크리스트 자동 평가 4가지:
  - 썸네일 기대를 넘는 임팩트 여부
  - 썸네일 문구를 도입부에서 반복했는지 (thumbnail callback)
  - 공감 장면이 앞에 있는지
  - 임팩트 장면을 앞으로 당겼는지
- 원고 욕구 자극 평가 (사고 싶은 충동이 드는지)
- 제목 개선안 (선택·후회·조건 적용)
- 썸네일 개선안 (결과·감정·공감 적용)

---

## 섹션 13 — 숏폼(릴스) 기획 원칙

인스타그램 알고리즘 핵심: 저장 > 공유 > 댓글 > 좋아요 순으로 노출에 영향

기획 원칙:
- 첫 1~3초 훅이 스크롤을 멈추게 해야 함
- 자막/텍스트 오버레이로 음소거 시청도 소화 가능해야 함
- 저장을 유도하는 "나중에 써먹을 수 있는 팁" 구조로 구성
- 루프 시청 유도: 영상 끝과 시작이 연결되게

---

## 섹션 14 — SNS 원소스 멀티유즈

유튜브 대본 1개 입력 → 3가지 자동 변환:

1. 블로그: 키워드 10회+ 반복, 2000자, 소제목 구조, SEO 최적화, 전화번호·자사몰 링크 자연스럽게 5회+ 삽입
2. 스레드: 5~7개 짧은 포스트, 첫 포스트 훅, 마지막 댓글 유도 질문
3. 숏폼 스크립트: 45~60초, 릴스·쇼츠·틱톡 공통 사용 가능

---

## 섹션 15 — 채널 성과 데이터 분석 항목

자체 채널 영상 데이터로 분석:
- 잘 되는 주제 vs 안 되는 주제 (데이터 기반 분류)
- 요일별·시간대별 평균 조회수 → 최적 업로드 타이밍 도출
- 영상 길이별 평균 조회수 (3분미만 / 3~8분 / 8~15분 / 15~30분 / 30분이상)
- 성공하는 제목 패턴 추출
- 채널 성장 병목 진단

---

## 디자인 요청사항

- 슬라이드 톤: 깔끔하고 전문적인 비즈니스 스타일
- 색상: 딥네이비 + 화이트 + 포인트 오렌지 또는 골드
- 각 슬라이드: 핵심 문장 1개를 크게 강조, 설명은 간결하게 불릿 포인트
- 표·다이어그램 활용: 섹션 3(콘텐츠 유형), 섹션 8(도입부 공식), 섹션 11(결과물 리스트)는 표나 흐름도로 시각화
- 표지 포함, 목차 슬라이드 포함
